# MODA - โปรแกรมพิมพ์ผ่าน WiFi และ Bluetooth

## การรันแอปพลิเคชันบน localhost

โปรเจคนี้ได้ถูกตั้งค่าให้สามารถรันบน localhost ได้แล้ว โดยใช้ IIS Express ซึ่งมาพร้อมกับ Visual Studio

### วิธีการรัน

1. เปิดโปรเจคใน Visual Studio
2. กด F5 หรือคลิกที่ปุ่ม "Start" (รูปสามเหลี่ยมสีเขียว) เพื่อรันแอปพลิเคชัน
3. Visual Studio จะเปิด IIS Express และรันแอปพลิเคชันบน localhost
4. เบราว์เซอร์จะเปิดขึ้นโดยอัตโนมัติพร้อมกับ URL ของแอปพลิเคชัน

### พอร์ตที่ใช้

- **HTTP**: IIS Express จะใช้พอร์ตที่ว่างอยู่ (เช่น 8080, 5000, หรือพอร์ตอื่นๆ)
- **WiFi Printer**: ใช้พอร์ต 9100 (ค่าเริ่มต้น) สำหรับการเชื่อมต่อกับเครื่องพิมพ์ผ่าน WiFi
- **Bluetooth Printer**: ใช้ Web Serial API ไม่จำเป็นต้องระบุพอร์ต

### การเข้าถึงหน้าต่างๆ

- หน้าหลัก: `http://localhost:[port]/`
- หน้าพิมพ์ผ่าน WiFi: `http://localhost:[port]/WifiPrint.aspx`
- หน้าพิมพ์ผ่าน Bluetooth: `http://localhost:[port]/BluetoothPrint.aspx`

### หมายเหตุ

- พอร์ตที่ใช้อาจแตกต่างกันไปในแต่ละครั้งที่รัน ให้สังเกตที่ URL ในเบราว์เซอร์
- หากต้องการกำหนดพอร์ตเอง สามารถแก้ไขได้ในไฟล์ `MODA.vbproj` และ `MODA.vbproj.user`
- การเชื่อมต่อกับเครื่องพิมพ์ผ่าน WiFi ต้องระบุ IP Address และพอร์ตของเครื่องพิมพ์ (ค่าเริ่มต้นคือ 9100)
- การเชื่อมต่อกับเครื่องพิมพ์ผ่าน Bluetooth ใช้ Web Serial API ซึ่งต้องรันบนเบราว์เซอร์ที่รองรับ (เช่น Chrome, Edge)
