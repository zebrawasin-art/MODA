Web Bluetooth BLE
วิธีใช้: Pair เครื่องพิมพ์ใน Android Settings → Bluetooth ก่อน
จากนั้นกด "เลือกเครื่องพิมพ์" — popup จะแสดงเฉพาะ device ที่ Pair แล้ว
📍 ดู MAC: Android Settings → Bluetooth → กดค้างชื่อเครื่องพิมพ์ → รายละเอียด
⚡ เมื่อบันทึกแล้ว ระบบจะค้นหาและเชื่อมต่ออัตโนมัติทุกครั้งที่เปิดหน้านี้
